Detailed Description

This class is responsible to split a string into segments terminated by one of a specified set of splitter characters. The terminating character is left as part of the previous string. Closing punctuation characters like quotes and brackets (and optional space between them) are kept with the previous chunk. Separators which occur before any non-white text are attached to the following sentence. Results are trimmed.
